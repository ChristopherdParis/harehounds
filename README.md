# HARE AND HOUNDS

## Configuracion
Python 3.10.12
Para instalar las dependencias ejecuta
```
pip3 install -r requirements.txt
```
Para ejecutar el juego
```
python3  main.py
```
Si no te funciona, no te funcionara
